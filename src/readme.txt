Windows: put slicer path in the path enviroment
    run by:
        command line: slicer --no-main-window --python-script path/to/smarkups.py
        double click: 1. copy SEEG.bat and smarkups.py to a folder.
                      2. right click SEEG.bat, select 'send to', 'desktop'
                      3. go to desktop, right click 'SEEG.bat - Shortcut', 'properties', change run to 'minimized'

Mac: put slicer path in the PATH enviroment
    run by:
        command line: slicer --no-main-window --python-script path/to/smarkups.py
        double click: 1. copy SEEG.sh and smarkups.py to a folder.
                      2. In finder, right click SEEG.sh, select "Open with" and then "Other...", select 'Terminal'. 
                         (To be able to select terminal you need to switch from "Recommended Applications" to "All Applications".)
                      3. create an alias of SEEG.sh, and drag it to the desktop.
